# Jinjer勤怠チェック拡張

> **見落としやすい勤怠ミスを、開いた瞬間に自動で見える化。**

`Jinjer勤怠チェック拡張` は、Jinjer の勤怠画面を自動チェックし、遅刻・打刻漏れ・休憩不足・集計不整合などを分かりやすく強調表示する Chrome 拡張です。  
対象ページ: `https://kintai.jinjer.biz/staffs/time_cards*`

---

## 主なメリット

- **見落とし防止**: 異常がある行やセルを色付きで表示し、確認漏れを減らします。
- **確認作業の時短**: 明細と合計値のズレを自動でチェックします。
- **操作性アップ**: シフト編集モーダルをドラッグ移動でき、画面が見やすくなります。
- **自分好みに調整可能**: ハイライト色は設定画面からカスタマイズできます。

---

## 使い方（3ステップ）

1. **拡張機能を読み込む**  
   Chrome の拡張機能ページで、この拡張を読み込みます。
2. **Jinjer の勤怠画面を開く**  
   対象 URL を開くと、自動でチェックが実行されます。
3. **ハイライト箇所を確認する**  
   異常のある行・セル・集計項目が強調されるので、そのまま修正や確認に進めます。

---

## 機能紹介

### 1. 勤怠の自動チェック

- **遅刻・早退・未打刻・欠勤・エラー** などを自動で検知
- **休憩不足 / 休憩超過** もチェック
- 有休で不足時間が補填されるケースは、誤検知を抑える判定に対応

### 2. 集計値の不整合チェック

- 画面下の合計値（サマリー）と明細行の再計算結果を比較
- ズレがある項目を強調表示し、月次確認時の見直しポイントを明確化

### 3. モーダルのドラッグ移動

- シフト編集モーダルをドラッグして任意の位置へ移動
- 背景の勤怠表を見ながら編集しやすくなります

### 4. 色のカスタマイズ

- ハイライト色は設定画面から変更可能
- チーム運用や個人の見やすさに合わせて調整できます

---

## インストール方法（開発者モード）

1. Chrome で `chrome://extensions/` を開く  
2. 右上の **「デベロッパーモード」** を ON  
3. **「パッケージ化されていない拡張機能を読み込む」** を選択  
4. 次のフォルダを指定: `/workspace/jinjer-extension`  
5. Jinjer の勤怠ページを再読み込みして、ハイライト表示を確認

## Q&A / 困ったときは

### Q. ハイライトが表示されません

- 対象ページ URL が `https://kintai.jinjer.biz/staffs/time_cards*` か確認してください。
- 拡張機能ページで、当該拡張が有効化されているか確認してください。
- 拡張を再読み込みしてからページを再読込してください。

### Q. 判定が合わないように見えます

- 表示文字列（ステータス名・休暇表記）が想定形式と異なると、判定結果に差が出ることがあります。
- 有休は `有休(◯h)` 形式でない場合、時間補填として扱えないことがあります。

### Q. アップデート後に動かなくなりました

- Jinjer 側の画面構造変更が原因の可能性があります。
- 最新版へ更新後も改善しない場合は、Issue で画面状況を共有してください。

## 技術仕様（必要な方向け）

- ビルド不要の Chrome 拡張（JS/CSS）
- 主な構成:
  - `jinjer-extension/manifest.json`
  - `jinjer-extension/content_script.js`
  - `jinjer-extension/src/scan-timecards.js`
  - `jinjer-extension/src/modal-drag.js`
  - `jinjer-extension/styles.css`
- 画面更新時は `MutationObserver` と `requestAnimationFrame` で再走査を集約
- バージョン: `0.0.1`
